1️⃣ Configurare utente e avatar

Aprire il file:

📁 _dati/configurazione.json

"nome_avatar": "Alice",
"nome_utente": "Emanuele",
"avatar_iniziale": "benvenuto",
"avatar_random": [ "sorridente", "soddisfatto" ],
"avatar_finale": "ciao.mp4",
"frase_finale": "Ci vediamo presto!",
"lingua": "it"

È consigliato personalizzare il nome dell’assistente e il vostro nome.

Esempio:
Se non lo fate, l’assistente dirà: "Ciao utente".

Tutte le immagini dell’avatar e il video sono modificabili nella cartella:

📁 _dati/asset/avatar/

Il campo lingua indica la lingua iniziale.
L’assistente rileva automaticamente tutte le lingue presenti nella cartella _dati.

  • lang_it.json
  • lang_en.json

2️⃣ Alias dei comandi

Configurabili nel file:

📁 _dati/configurazione.json

"dammi": [ "ricordami" ]

Se non serve un alias, lasciate vuoto:

"copia": []

Lista comandi con alias

"alias_comandi": {
  "dammi": [ "ricorda" ],
  "apri": [ "lancia" ],
  "aiuto": [ "help" ],
  "esci": [ "chiudi", "arrivederci" ],
  "ricorda": [ "registra" ],
  "modifica": [ "cambia" ],
  "elimina": [ "cancella" ],
  "cerca": [],
  "impara": [],
  "elenca": [],

  • Tutti gli altri devono averla

🌍 Aggiungere una nuova lingua

  1. Copia _dati/lang_en.json
  2. Rinomina ad esempio lang_fr.json
  3. Traduci i valori ma non le chiavi
  4. Imposta lingua_stt
  5. Riavvia l'assistente
