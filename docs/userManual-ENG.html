Personal Desktop Assistant – User Manual

Updated to release 2.2

Installation

The Assistant application is portable and can be launched from the desktop or a USB drive:
Download all necessary files from the Assistant GitLab project repository
Windows: download and unzip AssistenteVOICE_[last release].zip
Run assistente.exe
Linux: download and unzip AssistenteVOICE_[last release]_linux.zip
Run assistente
Python: download assistenteVOICE_[last release].py
Download the complete _dati/ folder
Make sure your Python installation has all required dependencies (see README)
Place assistenteVOICE_[last release].py and the _dati/ folder in the same directory
Run assistenteVOICE_[last release].py
For customization of the default language and other parameters, please refer to the Assistant .json Configuration Manual

Screenshots

Welcome screen


1) Area to enter commands. By pressing the microphone you can use voice commands
2) Command buttons area. If you remember the commands, you can close this sidebar using the arrow at the bottom right, next to the speaker
3) By clicking the arrow on the left of SHORTCUT, you can open/close your personalized shortcut commands (for configuration, please refer to the Assistant .json Configuration Manual)
4) Assistant answer area. On Windows, the answer will be spoken if the speaker is enabled. To stop the reading of the answer, enter the command ok

Language selection


If the language is not correct at startup, change it using the bottom-right buttons.
NOTE: the available languages are automatically detected by the application. To change the default language or add new ones, please refer to the Assistant .json Configuration Manual

TTS and STT


Note: in the Linux version, only Text-to-Speech is available and the loudspeaker button will be disabled.
