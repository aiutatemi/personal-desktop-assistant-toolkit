1️⃣ Configure user and avatar

Open the file:

📁 _dati/configurazione.json

"nome_avatar": "Alice",
"nome_utente": "Emanuele",
"avatar_iniziale": "benvenuto",
"avatar_random": [ "sorridente", "soddisfatto" ],
"avatar_finale": "ciao.mp4",
"frase_finale": "See you soon!",
"lingua": "en"

It is recommended to customize the assistant name and your name.

Example:
If you don’t, the assistant will say: "Ciao utente".

All avatar images and the video can be modified inside:

📁 _dati/asset/avatar/

The lingua field indicates the initial language.
The assistant automatically detects all languages present in the _dati folder.

  • lang_it.json
  • lang_en.json

2️⃣ Command aliases

Configurable in:

📁 _dati/configurazione.json

"dammi": [ "recall" ]

If you don’t need an alias, leave it empty:

"copia": []

List of commands with aliases

"alias_comandi": {
  "dammi": [ "ricorda" ],
  "apri": [ "launch" ],
  "aiuto": [ "help" ],
  "esci": [ "close", "bye" ],
  "ricorda": [ "record" ],
  "modifica": [ "change" ],
  "elimina": [ "erease" ],
  "cerca": [],
  "impara": [],
  "elenca": [],
  "copia": []
}

⚠ General notes

  • The last field inside { } and [] must not have a comma
  • All others must have one

🌍 Add a new language

  1. Copy _dati/lang_en.json
  2. Rename it (e.g. lang_fr.json)
  3. Translate values, not keys
  4. Set lingua_stt
  5. Restart the assistant
