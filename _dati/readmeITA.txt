* File di esempio per usare subito assistente personale desktop

** configurazione.json
modifica con le tue preferenze 

- Nome assistente, (consigliato personalizzare)
nome utente,  (consigliato personalizzare)
avatar iniziale, finale (opzionale)

- nomi aggiuntivi per i comandi (opzionale)

- risposte fisse (opzionale)

- shortcut (consigliato personalizzare per i comandi più usati)


** memoria.json
modifica oppure gestisci la memoria da dentro il programma dell'assistente
per memorizzare dati, lanciare programmi, pagine web o link a file e cartelle

** lang_it.json
file del linguaggio in italiano, modificabile per personalizzarlo (opzionale)

** lang_XX.json
file del linguaggio in altre lingue, modificabile per personalizzarlo (opzionale)
