Example files for immediate use of the desktop personal assistant

** configurazione.json
edit according to your system preferences

Assistant name (recommended to customize)
user name (recommended to customize)
initial and final avatar definitions (optional)

additional command aliases (optional)

predefined static responses (optional)

keyboard shortcuts (recommended to customize for frequently used commands)

** memoria.json
edit manually or manage memory directly from within the assistant application
used to store data, launch executables, open web pages, or reference files and directories

** lang_it.json
Italian language resource file, fully editable for localization or customization (optional)

** lang_XX.json
language resource file for additional locales, editable for customization (optional)
