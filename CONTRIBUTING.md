# 🤝 Help Welcome

Thank you for your interest in contributing to the project.

## 🍏 Prepare and Test Distributable Packages for macOS

If you’re working on macOS, you can help by creating and testing the installable packages.

📄 Detailed instructions: DocENGcreate-packeges.md
[Create Packages documentation](https://gitlab.com/EmanueleCAS/assistente/-/blob/master/docs/DocENGcreate-packeges.md)

---

## 🌍 Translate the Project into New Languages

If you speak another language, you can contribute by translating the lang_it.json file into your language and testing it directly in the assistant.

Check project languages folder: 
[Localization Files](https://gitlab.com/EmanueleCAS/assistente/-/tree/master/localization-file)

📄 Detailed instructions to create new file: DOCrel3.xLanguageSettings.md
[Language Settings documentation](https://gitlab.com/EmanueleCAS/assistente/-/blob/master/docs/DOCrel3.xLanguageSettings.md)

Once your translation is ready, please share the file with the project.

---

## 🧪 Test the Available Languages

All current languages have been tested, but real‑world usage is always the best way to spot issues and improve quality.

You can help by:

- using the assistant in your language
- correcting any mistakes you find
- improving command aliases to make them more natural
- sharing your updated file with the project
